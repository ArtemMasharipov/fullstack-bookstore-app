# Bookstore Frontend

Vue 3 + Pinia + Vuetify frontend for the bookstore application.

## Features

-   Modern Vue 3 Composition API
-   Pinia state management
-   Vuetify 3 UI components
-   Responsive design
-   JWT authentication
-   Shopping cart functionality
-   Admin panel

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
